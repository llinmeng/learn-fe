

# git 常用命令:


```bash

# 拉取一个新项目:
git clone git@github.com:llinmeng/learn-fe.git

# 拉取项目仓库最新代码:
git pull


# 查看代码仓库状态:
git status

# 把修改的文件, 添加到git版本控制
git add .          # 添加当前目录, 所有文件

git add xxx.xxx    # 添加一个指定文件
git add xxx/       # 添加一个指定目录


#添加提交信息.
git commit -m "update "


# 推送代码到github上.
git push


＃ 查看提交日志
git log


```
