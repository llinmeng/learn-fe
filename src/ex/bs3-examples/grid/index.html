Mixed: mobile and desktop

The Bootstrap 3 grid system has four tiers of classes: xs (phones), sm(tablets), md(desktops), and lg (larger deskteps). You can use nearly any combination of these to create more dynamic and flexible layouts.

Each tier of classes scales up, meaning if you plan on setting the same widths for xs and sm, you only need to specify xs.
